# 🚀 Organizr 2.0

**Organizr 2.0** is a modern **Task Management Web Application** built using **ASP.NET Core MVC** and **Entity Framework Core**. 
The project was developed to understand real-world web application development by implementing the complete MVC workflow, database management, authentication, and CRUD operations.

The application allows users to **register, log in, and manage their own personal tasks** in a secure environment. 
Each user has an individual dashboard where they can create, view, update, delete, and track the completion status of their tasks. 
The project also features a responsive UI, Bootstrap toast notifications, session-based authentication, and user-specific data management to provide a smooth user experience.

## ✨ Key Features

* 🔐 Secure User Registration & Login
* 👤 Session-Based Authentication
* 📝 Create, View, Edit & Delete Tasks
* ✅ Update Task Completion Status
* 📊 Personalized Dashboard
* 📱 Responsive & Modern Bootstrap UI
* 🛡️ User-Specific Task Access
